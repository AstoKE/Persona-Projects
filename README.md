# Persona Projects

Persona egitim programi kapsaminda gelistirilen projeler.

## Projeler

### 1. Web Gelistirme – JavaScript (`web-gelistirme-js/`)
React + Vite + Tailwind CSS ile gelistirilmis CRUD uygulamasi (Task Manager).
- Gorev ekle, listele, guncelle, sil
- Kategori ve oncelik filtreleme
- Netlify deploy: _link eklenecek_

**Calistirmak icin:**
```bash
cd web-gelistirme-js/my-crud-app
npm install
npm run dev
```

### 2. Mobil Uygulama – Flutter (`mobil-uygulama-flutter/`)
Flutter ile gelistirilmis Mini Katalog uygulamasi.
- Urun listeleme (GridView), arama, filtre
- Urun detay sayfasi (SliverAppBar)
- Sepet sistemi + satin al simulasyonu

**Calistirmak icin:**
```bash
cd mobil-uygulama-flutter/mini_katalog
flutter pub get
flutter run
```

### 3. Veritabani ve SQL (`veritabani-sql/`)
NovaStore e-ticaret veritabani projesi (SQL Server).
- 5 tablo: Categories, Products, Customers, Orders, OrderDetails
- DDL + DML + DQL sorgulari (JOIN, GROUP BY, SUM, DATEDIFF)
- VIEW ve BACKUP komutlari

**Calistirmak icin:**
SQL Server Management Studio'da `NovaStore_Proje.sql` dosyasini ac ve calistir.
